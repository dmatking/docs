# coder ssh

Enter a shell of execute a command over SSH into a Coder workspace

```text
coder ssh [workspace_name] [<command [args...]>]
```

### Examples

```text
coder ssh my-dev
coder ssh my-dev pwd
```

### Options

```text
  -h, --help   help for ssh
```
